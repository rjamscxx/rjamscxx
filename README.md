# Hi, I'm RJ 👋

Full-stack developer from the Philippines. Self-taught — I started by building a sales tracker for my own frozen-goods business because the notebook wasn't cutting it, and I haven't stopped building since. 16+ production apps later, this is what I do.

I build business software for real, small operations: POS systems, booking platforms, inventory, dashboards. Next.js + Postgres on the serious builds; single-file offline-first PWAs when that's what actually fits the owner's day.
